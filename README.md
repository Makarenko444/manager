# manager

Система для менеджеров интернет провайдера.

## Главная страница

В корне проекта размещены статические файлы `index.html` и `styles.css` с начальным макетом главной страницы сервиса manager.tomica.ru.

### Быстрый просмотр

Откройте `index.html` в браузере или поднимите любой статический веб-сервер:

```bash
python -m http.server
```

После запуска перейдите по адресу <http://localhost:8000>.
